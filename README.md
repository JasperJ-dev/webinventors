# Webinventors React App

Static React website for Webinventors, built with Vite. It does not require a WordPress backend.

## Run locally

Install Node.js 20.19 or newer, then run:

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production output will be generated in `dist/` and can be deployed to any static host.
